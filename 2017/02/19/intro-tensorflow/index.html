Tensorflow安装

虽说俺是个刚入门的游戏前端，但好歹还是对机器学习动过心，自认为也有足够的数学基础。借着一个机缘，开始捣鼓Tensorflow（google开源的深度学习框架），这里稍微记录一下安装中的重要点，一般一晚上可以弄好。

官方的详细安装步骤（有点儿问题）

确认系统和硬件配置

我的笔记本电脑：

  • 系统：windows 10 64位
  • 显卡：NVDIA GT940M （cuda compatibility 5.0，没问题，3.0以上就行了）
  • CPU：这个不重要，反正安装的都是GPU版（都说个人电脑用CPU跑太慢）

安装

  • 安装 cuda toolkit 8.0
    • CUDA 这里就不再多介绍了，参考以前的博文。
  • 安装 cuDNN
    • 需要注册 Accelerated Computing Developer Program 会员。
    • 解压后有三个文件夹，将其放在某个目录c:\xxx下，并把这个c:\xxx\bin添加到环境变量 PATH中。
  • 安装 anaconda，因为我选择采用anaconda里运行tensorflow而不是原生。选的是 anaconda 4.3.0 64bit (集成python 3.6)
  • 安装 tensorflow
    • 官网给出的方法在我的电脑上报了平台不匹配的错，参考这里的解答。其实就是限定虚拟环境为python 3.5

打开cmd

小试牛刀

最后再推荐一本tensorflow入门的书：《Tensorflow For Machine Intelligence》。
